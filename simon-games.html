Process

Developing Simon games on Arduino presents a challenge due to the requirement of completing the entire code within a single while loop. Furthermore, incorporating advanced rules such as timeouts and disregarding player inputs while the light signals are active adds another layer of complexity to the coding process.

Skills I gained

Through the APS160 course project, I delved into the world of electronic circuit design using TinkerCad simulation to create a Simon game. This hands-on endeavor involved programming an Arduino UNO to manage the game's logic, refining my microcontroller programming skills. As a result, I gained insights into crafting complex game rules, enhancing both my circuit design abilities and critical thinking. Moreover, I learned to optimize code for efficient execution, despite the challenge of containing it within a single loop. The project showcased my interdisciplinary aptitude, fusing electronics, programming, and game design. Overall, this experience empowered me with practical skills in simulation, coding efficiency, and logical thinking, applicable across various domains.
