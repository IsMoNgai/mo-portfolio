Discord Bot

Context

I helped a Minecraft Hypixel guild by developing a Discord bot to manage their server. Using Python exclusively, I coded the entire bot and utilized libraries like BeautifulSoup, pandas, and discord. Not being familiar with APIs at the time, I cleverly designed a Python crawler to extract data from the 'plancke' website. Knowing about APIs would have significantly reduced my development time for this project. Nonetheless, the experience provided an opportunity to apply my pandas knowledge effectively in data processing. xD

Process

The bot serves several purposes, like listing members and guilds, sending welcome messages, helping with role registration, offering guild info, and linking Discord names to in-game ones.

Fun Fact: While managing the member list, dealing with names containing "_" was tricky. Discord's automatic formatting sometimes made names like "_its_Mo" look italic as itsMo.

Skills I gained

Through the Minecraft Hypixel guild project, I cultivated a robust skill set. My Python programming proficiency flourished as I exclusively coded a comprehensive Discord bot, tapping into libraries such as BeautifulSoup, pandas, and discord. While the absence of API knowledge prompted innovative web scraping solutions, the underscore predicament underscored my attention to detail. Overall, this multifaceted journey underscored my resourcefulness, interdisciplinary adaptability, and a thirst for mastering new concepts.
